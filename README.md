# 🛡️ Turing-Pass (튜링 패스)
> 금융 보안의 새로운 표준: 행위 패턴 기반의 지능형 피싱 방지 솔루션

## 📂 프로젝트 소개
- **주제:** 컴퓨팅 사고 기반의 보이스피싱/메신저피싱 능동 방어 시스템
- **핵심 기술:** 사용자 행위 데이터(터치, 타자 속도, 앱 전환) 분석 및 동적 임계값 적용
- **기간:** 2025.12

## 🛠️ 주요 기능
1. **개인화된 기준점(Baseline):** 연령별/개인별 타자 속도 학습
2. **상황별 맞춤 방어:**
   - ⚠️ 앱 전환 과다 → 3분 지연 (Cooling-off)
   - 📞 통화 감지 → 차단 (Blocking)
   - 🐢 비정상 지연 → 본인 확인 (Turing Test)
